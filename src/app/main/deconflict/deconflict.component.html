Object Based Production

Deconfliction Example

This demonstration illustrates the application of TF/IDF (term frequency/inverse term frequency) analysis on a set of 90 movie descriptions from IMDB as compared with the same set of movies' descriptions from Wikimedia.

When the "Run Deconfliction" button is clicked, a similarity matrix is calculated between 40 randomly selected movies from each set. The movies with the lowest similarity score that exceeds the threshold are returned. The number of movie pairs that exceed the threshold is also shown.
